
## Description

Plant Shop is an API for browsing and purchasing plants. 

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server
  ```sh
   npm run start:prod
   ```
